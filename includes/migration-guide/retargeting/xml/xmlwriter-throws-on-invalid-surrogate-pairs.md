### <a name="xmlwriter-throws-on-invalid-surrogate-pairs"></a>XmlWriter вызывает на недействительные суррогатные пары

|   |   |
|---|---|
|Подробные сведения|Для приложений с целевой платформой .NET Framework 4.5.2 или предыдущих версий запись недействительной суррогатной пары с помощью обработки резервного исключения не всегда вызывает исключение. Для приложений, предназначенных для .NET Framework 4.6, попытка записи недействительной суррогатной пары вызывает <xref:System.ArgumentException?displayProperty=name>.|
|Предложение|При необходимости этот разрыв может быть счет, предназначенных для .NET Framework 4.5.2 или более ранней версии. Кроме того можно предварительно обработанные в допустимый xml до их записи недействительные суррогатные пары.|

|Версия|4.6|
